## About The Project

I started this as an experiment, automating some strategies I thought could be profitable on the altcoin exchanges. The current status of this project makes it hard to approach but I plan making prettifying the repository in future.
In case of interest, don't hesitate to contact me on mhavelka77@gmail.com


### Getting started

The bot can simply be run with 

```
python3 trading.py
```

All of the neccessary configuration resides in the `config.yaml` file. Currently, Bybit exchange API is implemented, but the interface under the `api` directory should make it easy to implement other exchange APIs.

The backtesting code can be invoked using the Jupyter notebook example located at `/backtest/playground.ipynb`

### Warning

**This bot is capable of executing trades after supplied with neccessary API keys, be sure you know what you are doing when using it**
